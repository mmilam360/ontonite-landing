# OnTonite Landing Page

Marketing landing page for [OnTonite](https://ontonite.app) — live music promotion for bars and restaurants.

## Stack

- Static HTML / CSS / JS — no framework
- Deployed on Vercel
- Google Fonts: Inter + Playfair Display

## Development

Open `public/index.html` in a browser. No build step required.

## Deployment

Connected to Vercel. Pushes to `main` auto-deploy to **ontonite.app**.

## Structure

```
public/
  index.html      — Full landing page
  styles.css      — All styles
  script.js       — Nav, FAQ accordion, scroll animations
  favicon.svg     — Browser favicon
  screenshots/    — Product screenshots (add real images here)
vercel.json       — Vercel deployment config
```

## Notes

- All "Sign Up" / "Start Free Trial" CTAs link to `https://manage.ontonite.app`
- Screenshot placeholders in `public/screenshots/` should be replaced with real product images
- Replace `public/logo.png` with actual brand logo when ready
